TinyBrain is an **API-first memory operating system** that gives AI agents long-term memory, context-aware intelligence, and the ability to learn from every interaction.

## Core Features

<AccordionGroup>
  <Accordion icon="brain" title="Semantic Memory System">
    Store and retrieve memories using AI-powered semantic search. Supports episodic, procedural, semantic, and identity memory types.
  </Accordion>
  
  <Accordion icon="bolt" title="100ms P95 Latency">
    Lightning-fast memory retrieval with guaranteed performance. All metrics are cryptographically verified.
  </Accordion>
